HTML
HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines elements like headings, paragraphs, links, images, and other components to build web pages

CSS
CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation and layout of HTML elements. It defines styles like colors, fonts, spacing, and positioning to make web pages visually appealing and responsive.

Javascript
JS (JavaScript) is a programming language used to create dynamic and interactive features on web pages. It enables functionality like form validation, animations, real-time updates, and user interactions.
